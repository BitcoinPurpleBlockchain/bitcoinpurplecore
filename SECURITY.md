# Security Policy

## Supported Versions

See our website for versions of BitcoinPurple Core that are currently supported with
security updates: https://bitcoinpurpleblockchain.com

## Reporting a Vulnerability

To report security issues, please use GitHub Security Advisory by clicking the "Security" tab in this repository.

When reporting, please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We take all security reports seriously and will respond as quickly as possible.
